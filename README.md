## Development

  - Install
  ```
  $ pip install -r requirements.txt
  ```

  - Run linter
  ```
  $ pylint app/
  ```

  - Use black (code formatter)
  ```
  $ black app/
  ```

  - Test
  ```
  $ pytest tests/
  ```
